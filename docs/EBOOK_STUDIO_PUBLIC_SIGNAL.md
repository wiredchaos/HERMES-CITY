## Status

HERMES-CITY tracks it only as a public-safe signal and coordination pattern.

```text
creator publishing need
  -> AGENTROPOLIS-CREATOR builds the template
  -> HERMES-CITY explains the public-safe use case
  -> AGENTROPOLIS-AGENT-MCP governs execution only if tooling becomes callable
```

## Public-Safe Description

Ebook Studio is a no-code browser builder for branded interactive ebooks.

A creator fills in a visual form, previews the ebook, and exports a single HTML file containing:

- branded cover text
- optional animated 3D cover scene
- chapter navigation
- reading progress
- completion checkmarks
- clean reader layout
- video explainer placeholders or embeds
- responsive phone, tablet, and desktop layout

## Boundary
