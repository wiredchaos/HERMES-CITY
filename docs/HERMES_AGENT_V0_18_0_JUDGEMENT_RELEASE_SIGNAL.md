# Hermes Agent v0.18.0: The Judgement Release

Public HERMES CITY signal note for Hermes Agent v0.18.0.

## Release Signal

Hermes Agent v0.18.0 is the Judgement Release.

The public signal is that Hermes is moving from a capable coding agent toward a broader orchestration system for learning, model councils, focused projects, and provider diversity.

## Major Capabilities

| Capability | Public Meaning | HERMES CITY Reading |
| --- | --- | --- |
| Mixture of Agents as a first-class virtual model | MoA becomes a native model surface instead of a side pattern. | Council-style intelligence can be routed as a first-class execution lane. |
| `/learn` | Agents can be taught from open-ended sources. | Knowledge acquisition becomes an operator-facing civic function. |
| Journey | Operators can visualize how an agent learned. | Learning paths become observable instead of hidden. |
| Gemini Vertex provider | Enterprise Google model infrastructure joins the provider map. | Provider diversity strengthens routing resilience. |
| Fable 5, Sonnet 5, and Fugu | More model options become available to the operator. | The model council inventory expands across capability lanes. |
| Desktop Projects support | Focused project spaces arrive in the desktop app. | Work can be scoped into district-like construction surfaces. |
| P0 and P1 issue clearance | Critical and high-priority issues were cleared. | Operational maturity matters as much as new capability. |

## Why This Matters

Agent-native systems cannot rely on a single model, a single memory path, or opaque learning behavior.

Hermes v0.18.0 strengthens the public case for:

- council-based inference
- teachable agents
- visual learning trails
- provider diversity
- project-scoped execution
- operational reliability

This is useful for HERMES CITY because public agent infrastructure needs to expose safe coordination patterns without leaking private implementation.

## HERMES CITY Position

HERMES CITY treats v0.18.0 as a high-signal release because it advances five core civic primitives:

1. **Judgement** through first-class Mixture of Agents.
2. **Learning** through `/learn`.
3. **Observability** through Journey.
4. **Resilience** through expanded provider support.
5. **Focused construction** through desktop Projects.
