# Hermes Serious Agent Stack

HERMES-CITY role: the Public Signal Layer.

This repo explains the serious Hermes agent stack in public-safe language. It should publish doctrine, diagrams, onboarding material, and clear examples without exposing private runtime details.

## Purpose

Make the Agentropolis operating model understandable while keeping the private city core separate.

## Seven lanes mapped to public signal

1. Brainstorming: teach design before implementation.
2. Soul Grader: explain identity before autonomy.
3. Project Memory: show repo memory patterns.
4. Agent Memory Wiki: explain curated memory over transcript dumping.
5. Safe Tool Wrapper: explain structured tool access and review gates.
6. Diagram Maker: publish clean architecture maps.
7. Skill Factory: show how repeated workflows become skills.

## Public boundary

HERMES-CITY may publish:

- doctrine
- diagrams
- public briefs
- onboarding docs
- safe examples
- signal writeups

HERMES-CITY should keep private:

- internal runtime details
- private roadmap details
- unreviewed operating procedures
- internal-only agent routes
- cross-realm private logic
