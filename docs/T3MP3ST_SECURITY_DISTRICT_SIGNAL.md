# T3MP3ST Security District Signal

## Public-safe signal

T3MP3ST is tracked as an authorized red team signal for HERMES CITY.

It should be presented as a security operations and cyber range capability, not as an unrestricted attack agent.

## District placement

```text
HERMES CITY
  -> Security District
     -> Red Team Lane
     -> Blue Team Lane
     -> Cyber Range Lane
     -> Patch Validation Lane
```

## HERMES role

HERMES routes the request.
HERMES does not grant permission by itself.
HERMES does not override law, ownership, written authorization, or scope.

## Public narrative

```text
Your agent does not need blind power.
It needs governed capability.

Security testing belongs behind scope, policy gates, validation, and receipts.
```
