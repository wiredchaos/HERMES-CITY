# HERMES CITY Architecture

HERMES CITY is the public civic shell for AGENTROPOLIS.

It is designed to communicate the coordination model without exposing private code, private prompts, production credentials, internal agent logic, wallet execution workflows, or proprietary infrastructure.

## Core relationship

```text
HERMES CITY = public interface
AGENTROPOLIS = private engine
```

## Public architecture

```text
Signal Sources
  -> RAILWATCH
  -> Nemotron Research Council
  -> Hermes Orchestrator
  -> NemoClaw Builder Lane
  -> Public Demo Interfaces
```

## Roles

### Hermes

Chief orchestration layer. Coordinates public signals, research briefs, module routing, and city-state alignment.

### NemoClaw

Builder and engineering lane. Turns approved ideas into public prototypes, docs, demos, and interface patterns.

### Nemotron

Research council. Compares evidence, ranks signal quality, and creates public-safe intelligence briefs.

### RAILWATCH

Trust and treasury OSINT layer for programmable money rails, agent commerce infrastructure, banking signals, credit unions, stablecoins, settlement rails, and tokenized treasury movement.

## Public/private boundary

Public content may include:

- Concepts
- Public schemas
- Diagrams
- Demo data
- Roadmaps
- Non-sensitive workflows
- Apache-2.0 reusable interface patterns

Private content must not include:

- Secrets
- API keys
- Client data
- Wallet keys
- Financial execution logic
- Internal prompts
- Agent orchestration internals
- Production deployment details
- Proprietary AGENTROPOLIS source code

## Design principle

Share the map. Protect the engine.
