# playhtml Public Presence Layer

## Purpose

This document describes a public, non-sensitive use of playhtml-style collaborative HTML spaces for HERMES CITY.

HERMES CITY is the public civic shell for Agentropolis concepts. It may show shared rooms, public maps, research boards, and safe interface patterns without exposing private Agentropolis implementation details.

## Public Use

playhtml can be used to prototype:

- public coordination rooms
- draggable ecosystem maps
- non-sensitive rail-watch boards
- agent-commerce education surfaces
- civic diagrams for settlement, wallet, and commerce coordination
- live public demo rooms for safe shared objects

## Boundary

Public room state must never contain:

- private keys
- credentials
- wallet secrets
- private Agentropolis implementation details
- client information
- proprietary prompts
- undisclosed strategy
- internal orchestration logic
- financial workflow internals

## Public Objects

| Object | Safe Public Role |
| --- | --- |
| Rail Signal Card | Shows public infrastructure signal |
| Ecosystem Node | Shows public company/protocol/category |
| Watch Board Item | Shows public research topic |
| Agent Commerce Card | Explains a public workflow concept |
| Civic Map Marker | Shows non-sensitive category placement |

## Design Rule

```text
HERMES CITY may show the map.
It must not expose the private engine.
```

## Suggested Demo

Build a public RAILWATCH room with draggable cards for:

- stablecoin rails
- banking signals
- credit union innovation
- retirement capital exposure
- agent wallet infrastructure
- commerce settlement rails

Each card should be educational, sourced, non-sensitive, and safe to share.
